## Execution Flow

| Step | Agent | Action |
|------|-------|--------|
| 1 | USER | Sends request |
| 2 | MILO | Receives, clarifies, translates, owns user relationship |
| 3 | ELON | Routes task - decides parallel vs sequential execution |
| 4 | CORTANA | Retrieves relevant context/memory |
| 5 | SPECIALISTS | Execute in parallel or sequence (Neo, Cornelius, Pulse, Hemingway, Jonny, Zuck, Sagan) |
| 6 | SENTINEL | QA gate - reviews all outputs |
| 7 | MILO | Delivers final output to user |
